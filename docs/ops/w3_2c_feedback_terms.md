## 実装

- **利用規約** web/terms.html (公開範囲・保存ポリシー表・ログの研究利用と
  「GTFS データの中身は研究に使わない」・CLI 誘導・出典/ライセンス・免責・
  削除依頼窓口)。実行ボタン脇に同意の一文 (簡易同意方式、決定 §6-2)
- **フィードバック**: ビューアの「この結果について問題を報告」(ホスト版のみ表示。
  入口からの報告は自動で**版固定 URL** に付け替え — 結果不変の原則により報告は
  常に再現可能)。POST /api/feedback → Feedback テーブル (RETAIN) + SES 自分宛て
  通知 (送信失敗でも記録は残る)
- **出典表示**: リポジトリ由来レポートの冒頭に
  「出典: GTFSデータリポジトリ {org}/{feed} — ライセンス: CC BY 4.0」(CC BY の
  表示義務を構造的に充足)
- **コストガード**: AWS Budgets 月 $20 (50%/100% 実績アラート → alert_email)。
  実績記録は docs/ops/costs.md
